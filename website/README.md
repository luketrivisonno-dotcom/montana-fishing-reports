# Montana Fishing Reports Website

Simple static website for the mobile app. Required for App Store submission.

## What's Included

- `index.html` - Landing page with app info
- `privacy.html` - Privacy Policy (REQUIRED by Apple)
- `support.html` - Support page with FAQ
- `terms.html` - Terms of Service

## Free Hosting Options

### Option 1: GitHub Pages (Recommended - Free)

1. Create a GitHub account: https://github.com/signup
2. Create a new repository named `montana-fishing-reports-website`
3. Upload these 4 HTML files to the repository
4. Go to Settings → Pages
5. Source: Deploy from a branch
6. Branch: main / root
7. Save
8. Your site will be at: `https://YOURUSERNAME.github.io/montana-fishing-reports-website`

### Option 2: Netlify Drop (Free)

1. Go to https://app.netlify.com/drop
2. Drag and drop these 4 HTML files
3. Get instant URL: `https://RANDOMNAME.netlify.app`
4. (Optional) Add custom domain later

### Option 3: Vercel (Free)

1. Go to https://vercel.com
2. Sign up with GitHub
3. Import this website folder
4. Get instant deployment

## Custom Domain (Optional)

If you want your own domain:
1. Buy domain at Namecheap, GoDaddy, etc. (~$10/year)
2. Point DNS to your hosting provider
3. Add domain in hosting settings

## Required URLs for App Store

After hosting, you'll have these URLs:

```
Privacy Policy: https://YOURSITE.com/privacy.html
Support: https://YOURSITE.com/support.html
Marketing: https://YOURSITE.com
```

## Before Using

Find and replace in ALL files:
- `[YOUR_EMAIL]` → your actual email
- `[YOUR_WEBSITE]` → your hosted URL
