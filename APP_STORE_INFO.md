## Description (4000 characters max)

### Short Version (for quick copy)
```
The #1 fishing app for Montana anglers. Get real-time river flows, water temperatures, fishing reports, and best times to fish for rivers across Montana.

FEATURES:
• 17 Montana Rivers: Madison, Yellowstone, Missouri, Gallatin, Bitterroot, and more
• Real-Time USGS Data: Current flow rates and water temperatures
• 7-Day Flow History: Visual charts showing trends
• Fishing Reports: Aggregated from top Montana fly shops
• Solunar Times: Best fishing windows based on moon phase
• Interactive Map: 78 USGS gauges + 300+ FWP access points
• Personal Fishing Log: Track your catches, flies, conditions
• Hatch Charts: Current hatches and recommended flies
• Offline Access: View cached data without internet

RIVERS COVERED:
Madison River, Yellowstone River, Missouri River, Gallatin River, Bitterroot River, Blackfoot River, Big Hole River, Beaverhead River, Bighorn River, Clark Fork River, Flathead River, Rock Creek, Ruby River, Stillwater River, Swan River, Jefferson River, Spring Creeks

WHY ANGLERS LOVE IT:
✓ Save time - all river data in one place
✓ Plan better - know conditions before you go
✓ Catch more - fish during optimal times
✓ Log catches - build your personal fishing history

Download now and fish smarter!

---

Note: Continued use of GPS running in the background can decrease battery life. Location data is used only to show nearby rivers and is not stored.
```

### Notes for Reviewer
```
This app aggregates publicly available fishing reports and USGS water data for Montana rivers. 

All data sources:
• USGS Water Services (waterdata.usgs.gov) - Public domain
• Montana Fish Wildlife & Parks (fwp.mt.gov) - Public access sites
• Individual fly shop websites - Public reports

No user account required. Location is optional and used only to show nearby rivers.

Test the app by:
1. Opening any river (e.g., Madison River)
2. Viewing flow data and fishing reports
3. Checking the interactive map
4. Adding a test catch to the fishing log
```
